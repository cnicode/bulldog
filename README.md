# bulldog
bulldog（ [ˈbʊldɔːɡ] 斗牛犬）是一款分布式链路日志跟踪监控框架，支持Openfeign、RestTemplate、Retrofit、HttpClient、OkHttp、RabbitMQ等方式透传链路信息。
